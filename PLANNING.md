## 1. 프로젝트 현황 분석

| 항목 | 기술 | 버전 |
|------|------|------|
| UI Framework | React | 18.3.1 |

| Build Tool | Vite | 5.4.2 |

### 1.3 현재 구현된 기능 (v1.0)

#### 사이드패널 AI 채팅
- AWS Bedrock Event Stream 실시간 스트리밍
- SigV4 서명 (Web Crypto API)
- 모델 선택 (Sonnet 4.6 / Opus 4.6 / Haiku 4.5)
- 대화 히스토리, 메시지 타임스탬프
- Auto-resize textarea, Enter/Shift+Enter

#### 대화 메모리 시스템
- CLAUDE.md 스타일 마크다운 메모리
- 시스템 프롬프트 자동 주입

- CRUD + append + 용량 표시

#### 작업 스케줄러
- chrome.alarms 백그라운드 실행
- 3가지 타입: 인터벌 / 매일 / 평일
- 브라우저 재시작 후 알람 자동 복원
- Chrome 알림 자동 생성

#### 에이전트 스웜
- Promise.all 병렬 멀티 에이전트
- 프리셋 팀: 리서치 (3에이전트) / 토론 (3에이전트)
- 커스텀 에이전트 추가
- 오케스트레이터 결과 종합

#### FAB 콘텐츠 스크립트
- 모든 웹페이지 우하단 Floating Action Button
- 6가지 컨텍스트 메뉴 (질문/요약/번역/다시쓰기/코드설명)
- 텍스트 선택 툴팁 (400ms)
- 다크모드 자동 감지

#### 팝업 런처
- 상태 대시보드 (API/작업/메모리)
- 5가지 빠른 동작
- 페이지 텍스트 추출

## 2. 비전 & 목표

### 핵심 가치

**"웹 브라우징 중 언제 어디서나 즉시 사용 가능한 AI 어시스턴트"**

- **즉시성** - 클릭 한 번으로 AI 활용, 별도 앱 전환 불필요
- **컨텍스트 인식** - 현재 웹페이지 내용을 자동으로 분석/활용
- **백그라운드 자동화** - 브라우저 종료 후에도 스케줄 작업 실행
- **서버리스** - 별도 백엔드 없이 AWS Bedrock 직접 연동

### hchat-pwa와의 시너지

| 구분 | hchat-pwa (웹앱) | hchat-extension (확장) |
|------|-----------------|----------------------|
| 접근성 | 별도 탭/앱 필요 | 모든 웹페이지에서 즉시 |
| 컨텍스트 | 수동 입력 | 페이지 자동 인식 |
| 기능 범위 | 120+ 기능 (풀스펙) | 핵심 기능 특화 |
| 자동화 | 세션 기반 | 백그라운드 스케줄러 |
| 저장소 | IndexedDB (무제한) | chrome.storage (10MB) |

### 타겟 사용자

1. **지식 근로자** - 웹 리서치, 문서 요약, 번역
2. **개발자** - 코드 리뷰, 기술 문서 분석, 디버깅 도우미
3. **콘텐츠 크리에이터** - 글쓰기 도우미, 브레인스토밍
4. **학생/연구원** - 논문 요약, 학습 노트 정리
